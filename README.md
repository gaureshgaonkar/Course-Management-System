# Course Management System (CMS)

Overview
A robust web application designed to efficiently manage educational workflows and institutional administration. This system facilitates seamless integration between courses, educators, and students, providing a centralized platform for academic tracking.

Key Features

Entity Management: Easily add, update, and manage comprehensive profiles for Students, Teachers, and Courses.

Dynamic Assignments: Allocate specific teachers to corresponding courses and systematically enroll students into their designated classes.

Attendance Tracking: An intuitive daily attendance module that allows educators to log and monitor student presence on a day-by-day basis.

Comprehensive Reporting: Access, view, and manage complete historical records for course enrollments, teacher assignments, and attendance logs.

Tech Stack

Backend: PHP(MVC Architecture)

Database: MySQL

Frontend: HTML, CSS, JavaScript
